==============
Tráta :tomato:
==============

::

  Tráta: Tomato.

A tiny CLI tool for Pomodoros

Install
=======

To install, run::

  $ make install

Usage
=====

To run in your CLI::

  $ trata

Here is a little preview of what you will see:

.. image:: ./assets/preview.png

Help
====

For help::

  $ trata --help
